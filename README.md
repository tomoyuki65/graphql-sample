# Next.js v15 + Apollo  
Next.js v15にApolloを導入し、GraphQLを試すサンプルです。  
  
## 機能一覧  
・GraphQL UI（/api/graphql）  
・ユーザー一覧ページ（/users）  
  
## 使用技術  
node           v20.18.0  
npm            10.8.2  
Next.js        15.0.2  
@apollo/client ^3.11.8  
@apollo/server ^4.11.2
